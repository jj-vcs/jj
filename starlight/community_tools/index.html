Community-built tools around Jujutsu

Many of these tools are not complete yet, just like Jujutsu itself. But they already simplify many workflows and can improve your experience.

Diffedit3 is a web-based alternate to Meld, as it no longer is packaged and available for all Distros. Its creator is also a frequent contributor.

GG - GUI for JJ

Section titled â€œGG - GUI for JJâ€

GG is a cross platform GUI for Jujutsu which makes all graph manipulating workflows quite easy. Take a look at its README.md as it quite descriptive.

Find it here.

Hunk.nvim

Section titled â€œHunk.nvimâ€

Hunk.nvim is a Neovim based diff-editor for Jujutsu which can be used as an alternative to the default :builtin diff-editor.

Find it here.

JJ-FZF

Section titled â€œJJ-FZFâ€

Centered around the jj log graph view, jj-fzf provides previews of diffs, the evolution-log, browses the op log and offers a large number of key bindings for commonly used jj operations from rebase to undo, and helps with divergent commits.

Find it here.

JJ TUI

Section titled â€œJJ TUIâ€

This is TUI for Jujutsu built in Ocaml, it is unopiniated and its creator is open to feedback.

Find it here.

Jujutsu Kaizen

Section titled â€œJujutsu Kaizenâ€

Jujutsu Kaizen is a plugin for Visual Studio Code. The goal of this extension is to bring the great UX of Jujutsu into the VS Code UI. Its developers are currently focused on achieving parity for commonly used features of VS Codeâ€™s built-in Git extension, such as the various operations possible via the Source Control view.

Find it here.

LazyJJ

Section titled â€œLazyJJâ€

lazyjj is a lazygit inspired TUI for Jujutsu.

Find it here.

Visual Jujutsu

Section titled â€œVisual Jujutsuâ€

VJJ is a fzf (fuzzy finder) wrapper for Jujutsu, which is meant to be used interactively in the terminal.

Find it here.

VisualJJ

Section titled â€œVisualJJâ€

VisualJJ is a plugin for Visual Studio Code which provides native integration for Jujutsu, not relying on Git colocation. Unlike other tools on this page, VisualJJ is not open-source.

Find it here.

Jujutsu UI

Section titled â€œJujutsu UIâ€

jjui is a terminal user interface for working with Jujutsu version control system.

Find it here.

Selvejj

Section titled â€œSelvejjâ€

Selvejj is a JetBrains plugin for integrating Jujutsu as a first-class VCS within the IDE.

Find it here.

PSCompletions

Section titled â€œPSCompletionsâ€

PSCompletions is a completion manager for a better and simpler tab-completion experience in PowerShell.

It can provide completions via psc add jj, or offer a better completion menu for the official completions.

Find it here.
